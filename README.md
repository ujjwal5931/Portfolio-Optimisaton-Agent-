# Portfolio Optimisation Agent

A Python-based intelligent portfolio management project that helps users analyze investments, optimize asset allocation, generate reports, and plan better financial decisions using modern portfolio theory concepts.

## Features

* Portfolio optimisation for better risk-return balance
* Investment planning and allocation suggestions
* Financial data fetching and processing
* Automated report generation
* Utility functions for portfolio calculations
* Simple frontend interface using static HTML

## Tech Stack

* Python
* HTML / CSS / JavaScript
* Financial Analysis Libraries
* Data Processing Tools

## Project Structure

## Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/ujjwal5931/Portfolio-Optimisaton-Agent-.git
```

2. Move into project folder

```bash
cd Portfolio-Optimisaton-Agent-
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python app.py
```
